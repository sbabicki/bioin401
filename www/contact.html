About the Author


Hi! My name is Sasha. Creating Heatmap App was my project for Bioinformatics 401 at the University of Alberta.
Information on what Heatmap App is can be found in the Instructions tab above. All source code for this project can be found at https://github.com/sbabicki/bioin401.

External Libraries and Sources

Some source code from other open source projects was used to create Heatmap App. Here is a complete list of these external resources:
Shiny

Heatmap App was built using Shiny by RStudio.

Gplots

The heatmap.2 function from the gplots library was used to generate the heatmaps in this application.

Bootstrap/Bootswatch

The CSS theme for Heatmap App was downloaded from bootswatch.com. The theme is called "Sandstone".

SpinKit

The spinner that is displayed while the page is loading was taken from SpinKit.

Shinyapps.io

Heatmap App is hosted on shinyapps.io by RStudio.
