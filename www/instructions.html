Overview of Heatmap App options menu features:

File Input

  • Can upload data in that separated using tabs, colons, or semicolons
  • txt or csv file extensions are accepted
  • Maximum upload size of 10 MB
  • Example files are offered
  • Example files can be downloaded

    • Analysis

  • Can choose clustering method: single, complete, average, or centroid linkage
  • Can apply clustering to rows and/or columns
  • Can choose to show row and/or column dendrograms
  • Can choose distance calculation method: euclidean or manhattan
  • Can choose scale type: row, column, or none

    • Image

  • Can change the height and width of the image preview on screen
  • Can change the heatmap colours
  • Can choose the number of shades of colours used for the heatmap
  • Can add a title, x axis label, and y axis label to the heatmap plot

    • Save

  • Can save a copy of the heatmap with the given image and analysis settings to your computer
  • Can choose to reset the height and width settings to an estimate by the app
  • Can adjust the resolution of the image
